# Sanity Content Entry Guide

Цей документ пояснює, в якому порядку та як вносити контент у Sanity CMS для сайту Дача TV. Дотримуйтесь порядку — деякі розділи залежать від інших.

---

## Доступ до Sanity Studio

Відкрийте Sanity Studio за адресою:
```
https://ваш-домен.vercel.app/studio
```
або локально — `http://localhost:3000/studio`.

Для входу потрібен обліковий запис Sanity (sanity.io). Якщо Studio не відкривається — перевірте, що змінна `NEXT_PUBLIC_SANITY_PROJECT_ID` встановлена у Vercel.

---

## Крок 1 — Налаштування сайту (Siteconfig)

**Де:** Studio → Налаштування сайту

Заповніть в першу чергу:

| Поле | Що вводити | Приклад |
|------|-----------|---------|
| Номер телефону | Лише ваш реальний телефон | `+380671234567` |
| Адреса (повна) | Для контактів та підвалу | `Коротич, Пісочинська ОТГ, Харківська обл.` |
| Telegram | Посилання на канал або особистий | `https://t.me/dachatv` |
| YouTube | Головна сторінка каналу | `https://youtube.com/@dachatv` |

Адреса (коротка) та соцмережі — за бажанням, після основного.

> Після збереження телефон одразу з'явиться в шапці та підвалі сайту.

## Крок 2 — Мед (Honey Products)

**Де:** Studio → Мед

Це найважливіший розділ. Сайт показує статичний заглушку, поки немає жодного продукту.

### Обов'язкові поля

| Поле | Опис |
|------|------|
| Назва | Повна назва. Приклад: `Мед Акація` |
| Slug | Натисніть **Generate** — заповниться автоматично |
| Сорт | Оберіть зі списку. Визначає текст і нотки на картці товару |
| В наявності | Вимкніть якщо сорту немає на складі |
| Порядок у каталозі | Число: 1 для першого в списку |

### Рекомендований порядок додавання

1. Акація (порядок: 1)
2. Липа (порядок: 2)
3. Сонях (порядок: 3)
4. Різнотрав'я (порядок: 4)
5. Сади (порядок: 5)
6. Ліс (порядок: 6)

### Фото

- Розмір: мінімум **800×800px**, квадрат
- Формат: JPG або WebP
- Завжди заповнюйте поле **Альтернативний текст** (для SEO)
- Приклад alt: `Мед акація 1L від пасіки Дача TV`

### Показати на головній

Поле **"Показати на головній"** (`isFeatured`) — увімкніть для 3–4 сортів. Якщо ви хочете контролювати, які саме відображаються, використайте "Налаштування головної сторінки" (крок 5).

### Пакування (необов'язково)

Натискайте Enter після кожного варіанту. Наприклад: `1L пластик` → Enter → `1L скло` → Enter → `3L пластик`

## Крок 3 — Відгуки (Reviews)

**Де:** Studio → Відгуки

### Як додавати

1. Введіть ім'я покупця (лише ім'я або ім'я + перша літера прізвища)
2. Місто
3. Текст відгуку — реальні слова, без редагування (окрім видалення особистих даних)
4. Оцінка — оберіть зірки
5. Поле **"Показувати на сайті"** — залиште вимкненим поки не перевірите відгук

> ⚠️ Відгуки не з'являться на сайті, поки ви не увімкнете "Показувати на сайті".

---

## Крок 4 — Продукти пасіки (Apinary Products)

**Де:** Studio → Продукти пасіки

Це пилок, прополіс, перга, горіхи в меду тощо. Заповнюйте після меду.

- Назва: `Пилок квітковий`, `Прополіс`, `Перга` тощо
- В наявності: вмикайте/вимикайте за потреби
- Порядок: число, менше = вище в списку

---

## Крок 5 — Для пасічників (Beekeeper Products)

**Де:** Studio → Для пасічників

Бджолопакети, бджолосім'ї, вулики. Якщо зараз не сезон — пропустіть, поверніться навесні.

Обов'язково вкажіть **Тип** (радіокнопки) — він визначає, в який розділ потрапить товар на сторінці.

Для бджолопакетів: заповніть поле **Породи** (наприклад: `Buckfast` → Enter → `Карніка`).

## Крок 6 — FAQ

**Де:** Studio → FAQ

Рекомендовані питання для старту:

**Категорія: Про продукти**
- Як зберігати мед?
- Чи натуральний ваш мед?
- Чому мед засахарився?

**Категорія: Замовлення**
- Як замовити мед?
- Яка мінімальна кількість для замовлення?

**Категорія: Доставка**
- У які міста ви доставляєте?
- Яка вартість доставки?

Завжди заповнюйте поле **Порядок** — це визначає, яке питання буде вгорі.

## Крок 7 — Налаштування головної (Homepage Config)

**Де:** Studio → Налаштування головної сторінки

Необов'язково, але корисно:
- **Рекомендовані сорти меду** — оберіть 3–4 конкретних продукти (посилання на продукти з Крок 2). Якщо не заповнено, сайт сам показує перші за порядком.

## Що перевірити після внесення контенту

- [ ] Сайт показує ваш телефон у шапці та підвалі
- [ ] Сторінка `/honey` показує ваші продукти, а не заглушку
- [ ] Кнопка "Замовити" відкриває форму (перевірте в мобільному браузері)
- [ ] Відгуки з'являються на головній (якщо є видимі)
- [ ] `/faq` показує ваші питання і відповіді
- [ ] `/studio` відкривається без помилок

## Часті питання

**Зміни не з'являються на сайті?**
Сайт кешує дані на 60 секунд (ISR). Зачекайте хвилину і оновіть сторінку. При розгортанні на Vercel кеш також оновлюється автоматично.

**Як видалити продукт?**
Знайдіть у Studio → натисніть три крапки → Delete. Але краще просто вимкнути "В наявності" — так продукт зникне з каталогу але залишиться в базі.

**Slug вже існує?**
Натисніть Generate знову або введіть унікальне значення вручну (латиниця, дефіси, без пробілів).
